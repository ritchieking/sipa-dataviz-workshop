Instructions

  1. Create a new Illustrator file by going to File > New. Name it 'Mobile-Banking' and set the profile to Web (which will set the units to pixels and the color mode to RGB). Then set the size to 640 x 480 (this will change the profile to "[Custom]", but never fear - the document will still use pixels and RGB). Click OK.

  2. Click on the Graph Tool and hold the button down until you see a menu pop up with different chart types. Select Bar Graph Tool.

  3. Draw a box with the Bar Graph Tool on your artboard. (Don't worry too much about the size — you can adjust that later). Paste in the mobile banking data, in the order that you want it to be in — regions first, followed by developing economies and high-income economies. You can paste the values for World in last... this ultimately won't be a bar, of course, but it makes it easier if you create one on the chart; later you can draw a line that is flush with the end of that bar and be confident that your line is in the right position.

  4. Pro-tip: Now you should make a copy of the chart and keep it off to the side of your artboard. That way, if one of the rectangles gets messed up at any point, you'll have a clean version to go back to.

  5. To make the chart look good, you need to ungroup it. With the chart selected, type Cmd+Shift+G (or go to Object >Ungroup). Don’t worry about the warning; that’s why we made a copy.

  6. Now that the chart is ungrouped, you can delete the axis, and resize the bars using the Selection Tool

  7. Go ahead and finish up the chart to make it match the chart at the top of the page (or mess around with your own styles and annotations, if you'd like), using these pointers:

    • Use the Type tool to add or edit text.

    • Use Arial and Arial Bold for now, and at least 13px font.

    • You can use the pen tool to draw little pointers to interesting things.

  8. Hint: to draw a line for the World overall, you can use the Pen tool. Follow the smart guides (in green) and click to draw a point that is lined up with the edge of the World bar, then hold down shift, move the mouse up and click again at the top of the Sub-Saharan Africa bar

  9. When you're done, you can export the image by going to File > Save For Web (or by contorting your hand to push Cmd+Opt+Shft+S)
